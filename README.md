# 골방 (Golbang)

> 너는 기도할 때에 네 골방에 들어가 문을 닫고 — 마태복음 6:6

기독교 개혁주의 신학 기반 회개·묵상 웹앱. 무료는 정적 페이지로 동작하고, 유료(동반자/동행)는 Claude API로 맞춤 묵상을 생성합니다.

## 파일 구조

```
golbang/
├── index.html        # 단일 페이지 (히어로 + 회개 앱 + 멤버십 + 간증 + FAQ)
├── styles.css        # 디자인 시스템 (warm ivory + ink + muted gold)
├── script.js         # 무료: 키워드 매칭 / 유료: /api/meditate 호출
├── api/
│   └── meditate.js   # Vercel 서버리스 함수 (Claude Sonnet 4.6 + 프롬프트 캐싱)
├── package.json
├── vercel.json
└── .env.example
```

## 로컬에서 실행

무료 기능은 백엔드 없이 정적으로 동작합니다.

```powershell
# 가장 간단: 정적 서버 하나 띄우기
npx serve .
# → http://localhost:3000 에서 확인
```

또는 `index.html`을 그대로 브라우저로 열어도 됩니다.

## 배포 (Vercel)

유료 멤버십의 맞춤 묵상 기능까지 사용하려면 Claude API를 호출할 서버가 필요합니다. Vercel 사용 시:

```powershell
npm install
vercel login
vercel env add ANTHROPIC_API_KEY   # 콘솔에서 발급한 키 붙여넣기
vercel deploy --prod
```

## 멤버십 활성화 (데모)

실제 결제 연동 전, 우측 멤버십 카드의 "동반자가 되기" 버튼을 누르면 데모 모드로 프리미엄이 활성화됩니다 (localStorage). 그 후 회개 입력란 아래 "맞춤 묵상으로 받기"를 체크하면 `/api/meditate`가 호출됩니다.

## 교회(B2B) 대시보드 — `church.html`

교회를 위한 멀티테넌트 관리 도구. 목회자가 교회를 만들고 초대코드로 교인을 모아 함께 씁니다.

**설정:** [CHURCH_SETUP.md](CHURCH_SETUP.md) 의 SQL을 Supabase에 1회 실행하면 켜집니다(16개 테이블 + RLS).

기능(탭 그룹):
- **홈** — 교인 수·오늘 생일·통독 진행·기도제목 요약, 최근 공지·다가오는 봉사
- **목양** — 교적·생일(검색·편집·CSV·생일 알림톡), 출석 체크인(셀프/관리자), 구역·셀, 새가족 양육 트랙
- **말씀** — 전교인 통독(순차/맥체인식 + 진도 현황), 큐티 나눔
- **예배** — 예배 콘티, 설교 노트(빈칸노트·QR 공유), 디지털 주보
- **소통** — 기도제목(중보 배정·댓글·내 중보 필터), 공지 보드(고정)
- **운영** — 봉사 스케줄, 헌금·재정 보고(투명 공개), 교회 설정·멤버 권한

**요금제:** 씨앗(개척·미자립 무료) / 등불 / 등대 / 성전 — [멤버십 페이지](membership.html#church)

**보안:** 모든 데이터는 Supabase RLS로 **교회 단위 격리**. 교인은 자기 교회 것만, 관리자만 교적·재정 등 관리.

**미연동(준비 중):** 알림톡 실발송(`api/notify.js` 스텁 — 발송대행사 키 필요), 온라인 헌금 결제(PG), 찬송가 가사(저작권).
